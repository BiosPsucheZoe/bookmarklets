Accessibility Bookmarklets are visualization tools that highlight hidden information that assitive technologies use to navigate through a webpage.

Once installed, these bookmarklets can interact with any webpage. To install a bookmarklet, drag and drop one into your browser's bookmarks bar.

  • Forms
  • Headings
  • Landmarks
  • Lists

Note: A bookmarklet is a JavaScript snippet that can be run within a web browser and that typically performs an action related to the currently loaded web page.
