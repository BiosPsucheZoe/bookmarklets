Accessibility Bookmarklets

Highlight accessibility features of web pages

  • Home
  • Installation
  • Documentation

To install the bookmarklets, drag and drop each one below into your browser's bookmarks or favorites bar.

  • Forms

    The Forms bookmarklet highlights the relationships between form actions, fields, and labels

  • Headings

    The Headings bookmarklet highlights all HTML5 headings on a given page

  • Landmarks

    The Landmarks bookmarklet highlights all ARIA landmarks and HTML5 tags that are interpreted as landmarks

  • Lists

    The Lists bookmarklet highlights every ordered list, unordered list, and data list on a given page, along with how many list items exist for each list

Done Installing?

Once installed, these bookmarklets can interact with any web page. Click it in your bookmarks or favorites bar to activate it. To de-activate, simply click it again or refresh your page; it will also de-activate when you resize your window or navigate to another page. See the documentation for more about using each bookmarklet.

Using a Different Browser?

The drag-and-drop installation works on modern desktop browsers including Chrome, Firefox, Safari, Opera, and IE 9 or above. The bookmarklets are not designed for installation on mobile devices such as on iOS, Android or Kindle.
